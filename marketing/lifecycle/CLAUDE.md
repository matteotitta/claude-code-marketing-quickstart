# marketing/lifecycle/ — your lifecycle lane

**Lifecycle**: email nurture, onboarding, retention, churn-prevention.

This lane follows the same arc as every execution lane: **research** → **strategy** → **execution**.

- [`research/`](./research/) — audit the current state, find gaps
- [`strategy/`](./strategy/) — plan, calendar, decide what to ship
- [`execution/`](./execution/) — the actual artifacts you ship

**Skills that produce content here:** `/lifecycle-marketing` (campaign design), `/email-nurture` (sequence writing).

**For the format:** see [`../../pulse-analytics-example/lifecycle/`](../../pulse-analytics-example/lifecycle/) — same structure, populated with PulseAnalytics examples.
