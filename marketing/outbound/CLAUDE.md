# marketing/outbound/ — your outbound lane

**Outbound**: cold email, ABM, prospecting.

This lane follows the same arc as every execution lane: **research** → **strategy** → **execution**.

- [`research/`](./research/) — audit the current state, find gaps
- [`strategy/`](./strategy/) — plan, calendar, decide what to ship
- [`execution/`](./execution/) — the actual artifacts you ship

**Skills that produce content here:** `/outreach-emails` (sequence writing), `/abm-campaign` (account-level campaigns), `/lead-scoring` (account prioritization).

**For the format:** see [`../../pulse-analytics-example/outbound/`](../../pulse-analytics-example/outbound/) — same structure, populated with PulseAnalytics examples.
