# marketing/competitors/ — your competitor research

This folder holds your **competitor analysis**: one file per competitor (named `MMYY-competitor-name.md`) plus an aggregate-insights file once you've analyzed 2+.

**To populate:** run `/competitor-research` against each competitor URL. Then `/competitor-aggregate` to roll N files into shared insights.

**For the format:** see [`../../pulse-analytics-example/competitors/`](../../pulse-analytics-example/competitors/).
